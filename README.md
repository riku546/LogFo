# LogFo (Log + Portfolio)とは

LogFoは、ソフトウェアエンジニアやプログラマーの目標実現を支援するプロダクトです。
目標とのギャップを可視化、日々の活動を記録、ポートフォリオを作成することで、目標実現を促進します。
