# Ubiquitous Language (ユビキタス言語)

開発チーム全員（エンジニア、デザイナー、PM、そしてAI）が共通認識を持つための用語集です。
コード内の変数名やクラス名は、可能な限りこの用語集に従ってください。

| 日本語 (Jp)  | 英語 (En / Code)          | 定義 / 備考 (Definition)                                            |
| :----------- | :------------------------ | :------------------------------------------------------------------ |
| **ユーザー** | `User`                    | システムを利用するアカウント主体。                                  |
| **ルーム**   | `Room`                    | ユーザーが集まり、チャットやログ共有を行う空間。                    |
| **ログ**     | `Log`                     | ユーザーが投稿する記録データ。                                      |
| **参加**     | `Join`                    | ユーザーがルームのメンバーになるアクション。                        |
| **退出**     | `Leave`                   | ユーザーがルームのメンバーから外れるアクション。                    |
| **管理者**   | `Admin` / `Owner`         | ルームの設定変更やメンバー管理権限を持つユーザー。                  |
| **認証**     | `Auth` / `Authentication` | ユーザーが誰であるかを確認すること (Login/Sign up)。                |
| **認可**     | `Authorization`           | ユーザーがリソースに対して何ができるかを確認すること (Permission)。 |

> [!NOTE]
> このリストは、プロジェクトの進行に伴い、適宜追加・更新してください。
